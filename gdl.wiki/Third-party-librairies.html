Third party librairies¶

Three general purpose and useful libraries are already packaged in Debian via the Debian Astro Project :

  • AstroLib, maintained by Wayne Landsman (idlastro official page at NASA). This library provides notably primitives to read and write FITS files.

  • Coyote by David W. Fanning (Coyote official page). High quality graphicals outputs and more.

  • MPfit by Craig Markwart (MPfit official page) The famous and convenient fitting procedure …

git clone https://salsa.debian.org/debian-astro-team/idlastro
git clone https://salsa.debian.org/debian-astro-team/coyote
git clone https://salsa.debian.org/debian-astro-team/mpfit

Many other libraries have been extensively tested and worked well with GDL 1.0.0+, including :

  • HEALpix by E. Hivon & al (HEALpix official page and download, comment on status of GDL 1.0.1)

  • Ulyss by Ph. Prugniel & al (Ulyss official page)

  • ATV by Aaron Barth (ATV official page) (wget https://www.physics.uci.edu/~barth/atv/atv.pro)

  • GRAFFER by James Tappin (GRAFFER official page)
