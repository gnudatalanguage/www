# Cloning gnudatalanguage/gdl

Do not use the 
![image](https://user-images.githubusercontent.com/33936193/213888017-95aab9bf-a0f6-4425-a9f8-b961c79224bc.png)
button of the GitHub repo, your cloning will be incomplete.

Starting from [#1484](https://github.com/gnudatalanguage/gdl/pull/1484) gdl uses **git submodules**.

To clone gdl, type: `git clone --recursive https://github.com/gnudatalanguage/gdl.git `

To update an already existing (cloned) gdl repo, do:
* `git submodule update --init` 
* or `git pull --recurse-submodules`
